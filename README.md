# Online Store - Coding Challenge for LTPLabs

A responsive online store built as a coding challenge.

Built with:

- React
- React Router
- TypeScript
- Tailwind CSS

---

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

---

## Notes

- Product data is fetched from the DummyJSON API which has some limitations
